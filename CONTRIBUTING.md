# 参与开发

本项目只接受与 Codex、飞书、个人微信三者连通直接相关的修复和改进。

提交前请：

1. 不提交 Token、App Secret、微信登录状态、日志和会话数据。
2. 为行为变更补充对应回归测试。
3. 执行 `make verify` 和 `make test-open-source-installer`。
4. 在 Pull Request 中说明改动、验证结果及是否影响一对一会话绑定。

问题与 Pull Request 请提交到本仓库：

- <https://github.com/yangzhousutpc-a11y/cc-connect-codex-sync/issues>
- <https://github.com/yangzhousutpc-a11y/cc-connect-codex-sync/pulls>
